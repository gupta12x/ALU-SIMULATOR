# 🔧 ALU Simulator in Scilab

## 📌 Project Overview

This project is an **Arithmetic Logic Unit (ALU) Simulator** implemented in **Scilab** as part of a *Computer Organization & Architecture* project.

The simulator performs basic arithmetic and logical operations based on a user-provided **opcode**, mimicking how an ALU works inside a CPU.

---

## ⚙️ Features

* Supports multiple ALU operations:

  * ➕ Addition
  * ➖ Subtraction
  * 🔗 Bitwise AND
  * 🔀 Bitwise OR
  * ⚡ Bitwise XOR
  * 🔄 Bitwise NOT (on input A)
* Interactive input from the user
* Simple and clean command-line interface
* Simulates 8-bit integer operations

---

## 🧠 ALU Operations Table

| Opcode | Operation | Description            |
| ------ | --------- | ---------------------- |
| 0      | ADD       | A + B                  |
| 1      | SUB       | A - B                  |
| 2      | AND       | Bitwise AND of A and B |
| 3      | OR        | Bitwise OR of A and B  |
| 4      | XOR       | Bitwise XOR of A and B |
| 5      | NOT       | Bitwise NOT of A       |

---

## ▶️ How to Run

1. Open **Scilab**
2. Copy and paste the code into a `.sce` or `.sci` file
3. Execute the file
4. Enter:

   * Integer A
   * Integer B
   * Opcode (0–5)
5. View the result in the console
